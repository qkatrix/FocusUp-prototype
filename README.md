# FocusUp - Produktivitātes lietotnes prototips

## Apraksts
FocusUp ir tīmekļa lietotne uzdevumu pārvaldībai un produktivitātes uzlabošanai. Prototips izstrādāts kā skolas projekts HTML, CSS un JavaScript tehnoloģijās.

## Funkcijas
- ✅ Uzdevumu pievienošana, rediģēšana, dzēšana
- ✅ Uzdevumu atzīmēšana kā izpildītus
- ✅ Detalizēta produktivitātes statistika
- ✅ Lietotāja profils ar personalizāciju
-✅ Tumšā/gaišā tēma
- ✅ Pilnībā adaptīvs dizains (darbojas uz telefoniem, planšetēm, datoriem)
- ✅ Datu saglabāšana lokāli (LocalStorage)

## Tehnoloģijas
- **HTML5** - semantiska struktūra
- **CSS3** - Grid, Flexbox, CSS mainīgie, Media Queries
- **JavaScript (ES6)** - lokālā datu glabāšana, DOM manipulācija
- **Font Awesome** - ikonas
- **Google Fonts** - Inter fonts

## Lietošanas instrukcija
1. Atver `index.html` jebkurā pārlūkprogrammā
2. Izmanto navigāciju, lai pārvietotos starp 5 lapām
3. Uzdevumu lapā pievieno savus uzdevumus
4. Statistiku lapā skaties savu progresu
5. Profila lapā iestadi savus datus
6. Iestatījumu lapā pielāgo lietotni

## Lapas
1. **Sākumlapa** (`index.html`) - apskats un iespējas
2. **Uzdevumi** (`tasks.html`) - uzdevumu pārvaldība
3. **Statistika** (`statistics.html`) - progresu grafiki
4. **Profils** (`profile.html`) - lietotāja konts
5. **Iestatījumi** (`settings.html`) - lietotnes iestatījumi
